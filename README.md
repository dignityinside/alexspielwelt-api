# Alex Spielwelt API

API for [Alex Spielwelt Frontend](https://github.com/dignityinside/alexspielwelt-frontend)

Made with ❤️ in [NestJS](https://nestjs.com/)

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```

#### Development links

```
API enpoint: http://localhost:3001
Adminer: http://localhost:8080
Swagger: http://localhost:3001/api
```

#### Build and run Docker

- `docker-compose up` - Start Docker

Docker database credential:

```
System: PostgreSQL
Server: postgres
Username: root
Password: password
```
